普通模式

## 字符与音高对应关系

本项目使用 26 个英文字母对应 26 个音源文件。

当前采用的是钢琴式映射：

```text
低音1-7 = a b c d e f g
中音1-7 = h i j k l m n
高音1-7 = o p q r s t u
倍高音1-5 = v w x y z
```

展开如下：

| 字符 | 对应音级 |
|---|---|
| `a` | 低音1 |
| `b` | 低音2 |
| `c` | 低音3 |
| `d` | 低音4 |
| `e` | 低音5 |
| `f` | 低音6 |
| `g` | 低音7 |
| `h` | 中音1 |
| `i` | 中音2 |
| `j` | 中音3 |
| `k` | 中音4 |
| `l` | 中音5 |
| `m` | 中音6 |
| `n` | 中音7 |
| `o` | 高音1 |
| `p` | 高音2 |
| `q` | 高音3 |
| `r` | 高音4 |
| `s` | 高音5 |
| `t` | 高音6 |
| `u` | 高音7 |
| `v` | 倍高音1 |
| `w` | 倍高音2 |
| `x` | 倍高音3 |
| `y` | 倍高音4 |
| `z` | 倍高音5 |


## rhythm 写法

| 写法 | 含义 |
|---|---|
| `h/1` | `h` 音，全音符 |
| `h/2` | `h` 音，二分音符 |
| `h/4` | `h` 音，四分音符 |
| `h/8` | `h` 音，八分音符 |
| `h/16` | `h` 音，十六分音符 |
| `h/4.` | `h` 音，附点四分音符 |
